Cara Export Aset

untuk Figma / Penpot / Canva
📋 Copy Token CSS

Buka styles.css, copy bagian :root { ... } ke Figma Tokens plugin atau Style Dictionary.

🖼️ Export SVG Icon

Buka DevTools → pilih elemen SVG → klik kanan → Edit as HTML → copy kode SVG-nya → simpan sebagai nama.svg.

🆓 Figma Free Tier

Figma gratis untuk 3 file aktif. Cukup untuk 1 project SMK. Buat akun di figma.com.

🐧 Penpot (100% Gratis)

Open-source, tidak ada batas file. Support import SVG & CSS tokens. Buka di penpot.app.
